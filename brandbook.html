06 — Reglas

Do's & Don'ts

Principios para mantener la coherencia visual de la marca.

Hacer
  • Usar fondos oscuros navy como base (dark-first)
  • Aplicar Poppins en headings y Lato en body
  • Usar teal (#00bfa5) para CTAs y highlights
  • Diseñar mobile-first siempre
  • Usar gradientes navy→blue y blue→teal
  • Aplicar glassmorphism para efectos de profundidad
  • Usar Lucide Icons (línea limpia)
  • Mantener máximo 2 colores de acento por componente
  • Comunicar siempre en español, tono cercano
No hacer
  • Usar fondos blancos sin propósito claro
  • Usar Inter, Roboto u otras fuentes genéricas
  • Usar colores fuera de la paleta sin justificación
  • Diseñar solo para desktop
  • Mezclar más de 2 acentos en un componente
  • Usar iconos de estilos distintos a Lucide
  • Omitir el hover state en elementos interactivos
  • Usar pesos de fuente incorrectos (body en bold)

Referencia Rápida — Tailwind CSS

/* Colores */
bg-[#0a2342]          // Navy — fondo principal
bg-[#1d70a2]          // Blue — secundario
text-[#00bfa5]        // Teal — acento, CTAs
text-[#e0e0e0]        // Gray — texto secundario
border-[#1d70a2]/30   // Blue border 30% opacity

/* Tipografía */
font-poppins font-bold        // Headings
font-lato text-[#e0e0e0]     // Body
text-6xl lg:text-8xl         // H1 Hero
text-4xl lg:text-6xl         // H2 Sección

/* Gradientes */
bg-gradient-to-br from-[#0a2342] to-[#1d70a2]
bg-gradient-to-r from-[#1d70a2] to-[#00bfa5]
bg-gradient-to-r from-[#1d70a2] to-[#00bfa5] bg-clip-text text-transparent

/* Card estándar */
bg-gradient-to-br from-[#0a2342] to-[#1d70a2] rounded-lg p-6 border border-[#1d70a2]/30 shadow-xl
