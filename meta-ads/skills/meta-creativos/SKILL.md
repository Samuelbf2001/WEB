# Meta Creativos — Briefs y Copy para Anuncios

En 2026, el creativo ES el targeting. El sistema Andromeda/GEM de Meta usa los creativos para determinar a quién mostrar los anuncios. Un creativo potente amplifica el targeting; uno débil desperdicia el presupuesto.

## Información necesaria

```
Para generar los creativos necesito:

1. ¿Qué se está anunciando? (producto/servicio específico, precio, oferta)
2. ¿Cuál es el objetivo del anuncio? (venta directa, lead, mensaje, tráfico)
3. ¿A quién va dirigido? (avatar del cliente ideal)
4. ¿Cuál es el principal dolor o problema que resuelve?
5. ¿Cuáles son los 3 beneficios más importantes?
6. ¿Hay alguna oferta, garantía o urgencia disponible?
7. ¿Qué formatos tienen disponibles? (video, imagen, pueden producir Reels)
8. ¿Tienen testimoniales o casos de éxito reales?
9. ¿Cuál es el tono de marca? (formal, cercano, aspiracional, informativo)
```

## Framework de Copy: HOOK → PROBLEMA → SOLUCIÓN → PRUEBA → CTA

### Estructura universal de anuncio de conversión

```
[HOOK — primeros 3 segundos o primera línea]
Captura la atención específica del avatar.
Debe generar: "esto es para mí"

[PROBLEMA/AGITACIÓN]
Describe el dolor de forma específica.
Usa las palabras exactas que usaría el cliente.

[SOLUCIÓN]
Presenta el producto/servicio como la respuesta.
Enfócate en beneficios, no en características.

[PRUEBA SOCIAL / CREDIBILIDAD]
Testimonial, número de clientes, resultado específico.
"Más de X clientes ya lo hicieron"

[CTA — llamada a la acción]
Una sola acción clara y específica.
```

## Tipos de hooks — Los más efectivos para LATAM

### Hooks por dolor/problema
```
"¿Cansado/a de [problema específico]?"
"El [problema] es más común de lo que crees"
"Si [situación problemática], esto te interesa"
"Por qué el [método tradicional] ya no funciona"
"El error que cometen el [X]% de [avatar]"
```

### Hooks por resultado/transformación
```
"Cómo [resultado deseado] en [tiempo específico]"
"De [situación inicial] a [situación final] en [tiempo]"
"Lo que nadie te dice sobre [tema relevante]"
"El secreto de [personas que lograron el resultado]"
"Así consiguió [persona real/tipo] [resultado específico]"
```

### Hooks de oferta/urgencia
```
"Solo hasta [fecha]: [oferta]"
"Últimos [N] cupos disponibles"
"[Descuento]% de descuento por tiempo limitado"
"Gratis por primera vez: [beneficio de entrada]"
"Precio especial solo para [ciudad/segmento]"
```

### Hooks de curiosidad
```
"La verdad sobre [tema de interés del avatar]"
"Nadie habla de esto en [industria]"
"[Número] razones por las que [afirmación contraintuitiva]"
"Antes de [acción que ya van a tomar], lee esto"
```

## Templates de copy completo

### Template 1 — Venta directa (producto)
```
[HOOK de dolor o resultado]

[Ampliar el problema — 1-2 líneas sobre la situación que vive el avatar]

✅ [Beneficio 1 — específico y concreto]
✅ [Beneficio 2 — diferenciador]
✅ [Beneficio 3 — prueba o garantía]

[Resultado esperado o transformación]

👉 [CTA específico] → [URL o acción]

[Urgencia o escasez si aplica]
```

**Ejemplo — E-commerce ropa:**
```
¿Cansada de que tu ropa no te dure ni 3 lavadas?

La fast fashion está diseñada para romperse. Nosotros no.

✅ Telas de calidad premium certificada
✅ Costuras reforzadas para durar años
✅ Si no te enamoras, te devolvemos el dinero

Prendas que duran tanto como tus memorias.

👉 Ver colección nueva → [link]

Envío gratis hoy en Bogotá, Medellín y Cali.
```

### Template 2 — Generación de leads (servicio)
```
[HOOK — pregunta directa al avatar]

[Problema específico que enfrentan — 1 párrafo]

[Lo que logran con el servicio — 2-3 bullets con resultados]

[Credencial o prueba social — 1 línea]

👉 [CTA: Agenda / Solicita / Habla con]
[Subtext: sin costo, sin compromiso, gratis]
```

**Ejemplo — Consultoría financiera:**
```
¿Ganas bien pero no sé a dónde se va el dinero?

El 73% de los colombianos llegan a fin de mes con saldo en rojo, no por ganar poco, sino por no tener un sistema.

En solo 1 sesión de diagnóstico:
→ Identificamos tus fugas de dinero
→ Diseñamos tu plan de ahorro real
→ Te damos 3 acciones para implementar esta semana

Más de 500 familias colombianas ya ordenaron sus finanzas con nosotros.

👉 Agenda tu diagnóstico gratis
(30 minutos, sin costo, sin compromiso)
```

### Template 3 — WhatsApp/Mensajes
```
[HOOK directo y conversacional]

[Problema en lenguaje cercano — como si lo dijera un amigo]

[Qué ofrecen en el chat — expectativa del usuario]

💬 [CTA de conversación]
[Lo que pasa cuando escriben]
```

**Ejemplo — Servicio médico:**

```
¿Llevas meses con ese dolor y no sabes qué hacer?

No tienes que aguantarte. Muchas veces una consulta lo resuelve.

Escríbenos y te damos:

💬 Escríbenos ahora

```

### Template 4 — Retargeting (audiencia cálida/caliente)
```
[Referencia a la interacción previa — sutil]

[Recordatorio del beneficio principal]

[Manejo de objeción más común]

[Urgencia o incentivo para actuar ya]

👉 [CTA directo]
```

**Ejemplo — E-commerce:**
```
Viste estos [productos]. Todavía están disponibles.

[Imagen del producto que vio]

¿Dudando por el precio? Hoy tienes 10% de descuento exclusivo.

Código: VUELVE10

Solo válido en las próximas 24 horas.
👉 Completar mi compra
```

### Template 5 — Video/Reel (script de 30-60 segundos)
```
[SEGUNDO 0-3: Hook visual o verbal IMPACTANTE]
Lo más importante: detener el scroll.

[SEGUNDO 3-10: Problema o situación relatable]
El usuario se identifica.

[SEGUNDO 10-25: Solución / demostración]
Mostrar, no solo decir.

[SEGUNDO 25-40: Prueba social / resultado]
Credibilidad rápida.

[SEGUNDO 40-60: CTA claro]
Una sola acción.
```

**Script de ejemplo — Curso online:**
```
[0-3s]: "Este error le cuesta a los emprendedores millones cada mes"
[Texto en pantalla: "$$ perdidos por no hacer esto"]

[3-10s]: "La mayoría de dueños de negocio gasta en publicidad sin saber qué está funcionando. Pautan a ciegas."

[10-25s]: "En nuestro curso aprendes a leer tus métricas en 2 horas y a tomar decisiones con datos. Cero suposiciones."

[25-40s]: "Más de 300 emprendedores colombianos ya multiplicaron su ROAS con este método."

[40-60s]: "Enlace en la bio. Cupos limitados para esta semana."
[CTA visual: "Ver curso ahora →"]
```

## Guía de tono por industria

| Industria | Tono | Palabras clave | Evitar |
|-----------|------|----------------|--------|
| Moda/Belleza | Aspiracional, cercano | "descubre", "transforma", "eres" | Tecnicismos |
| Finanzas | Confiable, directo | "seguro", "crece", "protege" | Promesas exageradas |
| Salud | Empático, informativo | "bienestar", "cuidar", "mejorar" | Urgencia agresiva |
| Tecnología | Moderno, solucionador | "optimiza", "automatiza", "ahorra tiempo" | Jerga técnica |
| Educación | Motivacional, práctico | "aprende", "avanza", "transforma" | Lenguaje académico |
| B2B | Profesional, orientado a ROI | "resultados", "eficiencia", "crecer" | Informalidad excesiva |

## Diversidad de creativos — Regla Andromeda

Meta's Andromeda agrupa creativos similares (> 60% similitud visual/textual) y reduce el alcance de los duplicados. Para evitar supresión:

```
MÍNIMO por ad set: 3 creativos
RECOMENDADO: 5-7 creativos

Variaciones necesarias:
□ Al menos 2 formatos diferentes (ej: video + imagen)
□ Al menos 2 hooks diferentes
□ Al menos 2 propuestas visuales distintas
□ Mix de video corto (< 15s) + video largo (30-60s)
□ UGC (user-generated) vs. producción profesional

Señales de fatiga creativa (pausar y renovar):
- Frecuencia > 3.0 en una semana (prospección)
- CTR cae > 30% respecto a semana anterior
- CPM sube > 40% sin cambios de audiencia
- CPA sube > 30% sin cambios de puja
```
