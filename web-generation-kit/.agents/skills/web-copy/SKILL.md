# Web Copy

Redactar texto claro, especifico y orientado a conversion para la empresa del proyecto actual.

## Base

Leer `context/PROJECT.md` si existe. Si falta informacion, inferir con cuidado desde el pedido y evitar claims no comprobados.

## Voz

- Espanol latinoamericano neutro por defecto.
- Profesional, directo y humano.
- Concreto antes que grandilocuente.
- Evitar jerga, frases infladas y promesas imposibles.

## Hero

- H1: 6 a 10 palabras cuando sea posible.
- Debe expresar resultado, categoria u oferta.
- El subtitulo amplifica el H1, no lo repite.
- CTA principal con verbo de accion y resultado.

Ejemplos de estructura:

```text
[Resultado claro] para [audiencia especifica]
[Verbo] [problema] con [mecanismo diferenciador]
```

## CTAs

- 2 a 5 palabras.
- Verbo claro.
- Evitar "Ver mas", "Click aqui" y "Enviar" salvo que el contexto lo justifique.

Patrones:

```text
Agenda una llamada
Solicita una demo
Calcula tu plan
Empieza ahora
Descarga la guia
```

## Servicios

Estructura recomendada:

1. Que es.
2. Para quien.
3. Resultado.
4. Siguiente accion.

## SEO

- Meta title: 50 a 60 caracteres, keyword cerca del inicio.
- Meta description: 145 a 160 caracteres, beneficio + CTA.
- No repetir keywords de forma artificial.

## Checklist

- El texto dice algo especifico de esta empresa.
- El beneficio esta antes que la caracteristica.
- El CTA indica una accion concreta.
- No hay claims imposibles de verificar.
- No hay relleno tipo "soluciones innovadoras", "lideres del mercado" o "de clase mundial".
